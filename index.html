Get started in 5 steps
Set up GA4 tracking on your pages, then connect this dashboard
01

Create a Google Analytics 4 property

Go to analytics.google.com → Admin → Create Property. Under Data Streams, add a Web stream for each page you want to track.

02

Add the tracking snippet to each page

Each stream gives you a Measurement ID like G-XXXXXXXXXX. Use the snippet generator in Configure to get the complete ready-to-paste code.

03

Enable the GA4 Data API in Google Cloud

Go to console.cloud.google.com → APIs & Services → Library → search "Google Analytics Data API" → Enable it.

04

Create OAuth 2.0 credentials

Google Cloud → Credentials → Create Credentials → OAuth 2.0 Client ID. Application type: Web application.

Add to Authorized JavaScript Origins:

https://analytics.wetrademarketing.com/

Also add your email as a Test User under OAuth consent screen.

05

Set up sharing (for colleagues)

Go to jsonbin.io → sign up free → Create Bin → paste {} → Save. Copy your Bin ID and Master API Key.

Paste both into the Sharing section in Configure, set a password, and save. Then click the Share button to copy the link for colleagues.
