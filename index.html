Get started in 5 steps
Set up GA4 tracking on both pages, then connect this dashboard
01

Create a Google Analytics 4 property

Go to analytics.google.com → Admin → Create Property. Name it something like "WeTrade Campaigns". Choose your time zone and currency (MYR or USD).

Under Data Streams, add a Web stream for each page:

leaderboard.wetrademarketing.com milliondollar.wetrademarketing.com

💡 You can use one property with two data streams — simpler to manage.

02

Add the tracking snippet to each page

Each stream gives you a Measurement ID like G-XXXXXXXXXX. Paste this inside the <head> tag of each page:

<!-- Google tag (gtag.js) --> <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'G-XXXXXXXXXX'); </script>

Replace G-XXXXXXXXXX with each page's actual Measurement ID. Use the snippet generator in the Configure tab once you have your IDs.

03

Enable the GA4 Data API in Google Cloud

Go to console.cloud.google.com → Create or select a project → APIs & Services → Library → search "Google Analytics Data API" → Enable.

04

Create OAuth 2.0 credentials

In Google Cloud Console → APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID.

Application type: Web application. Under Authorized JavaScript Origins, add:

http://localhost http://127.0.0.1 file://

Copy the Client ID — you'll paste it in the Configure tab.

⚠️ If you open this file directly from your filesystem (file://), also add null to the origins list in Google Cloud Console.

05

Find your GA4 Property ID

In Google Analytics → Admin → Property Settings. The Property ID is a number like 123456789. Note it down — you'll need it in Configure.

⏱ GA4 takes 24–48 hours to start showing data after you add the snippet. Real-time data is available immediately under Reports → Real-time in Google Analytics.
